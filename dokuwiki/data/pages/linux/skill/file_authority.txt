====== 只更改目录权限 ======
只更改目录以及所有子目录权限，不改变目录内文件权限。

<code bash>
find -type d -exec chmod 750 {} \;
</code>

====== 只更改文件权限 ======

只更改目录以及子目录内文件权限，不改变目录权限。 

<code bash>
find -not -type d -exec chmod 640 {} \;
</code>

======以上两个操作如果修改的文件夹过多的话会报错======

====== 只更改目录权限 ======
只更改目录以及所有子目录权限，不改变目录内文件权限。

<code bash>
find ./ -type d |xargs chmod 750
</code>

====== 只更改文件权限 ======

只更改目录以及子目录内文件权限，不改变目录权限。 

<code bash>
find -not -type d |xargs chmod 640
</code>
