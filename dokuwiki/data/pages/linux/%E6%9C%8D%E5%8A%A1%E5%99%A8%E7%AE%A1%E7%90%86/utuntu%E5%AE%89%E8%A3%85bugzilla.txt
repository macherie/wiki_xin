======Utuntu安装Bugzilla======

=====1. 安装Apache=====

=====2. 安装MySQL=====

新建数据库：
<code>
create database bugs;
use bugs;
GRANT SELECT, INSERT, UPDATE, DELETE, INDEX, ALTER, CREATE, LOCK TABLES,CREATE TEMPORARY TABLES, DROP, REFERENCES ON bugs.* TO bugs@localhost IDENTIFIED BY "123";
FLUSH PRIVILEGES;
</code>

=====3. 安装Bugzilla=====

解压bugzilla-4.4.2.tar.gz到/var/www，重命名为bugzilla

=====4. 安装perl模块=====

配置CPAN

运行perl -MCPAN -e shell
<code>
Would you like to configure as much as possible automatically? [yes]

What approach do you want?  (Choose 'local::lib', 'sudo' or 'manual')

 [local::lib] 

Would you like me to automatically choose some CPAN mirror

sites for you? (This means connecting to the Internet) [yes]no

Would you like to pick from the CPAN mirror list? [yes]no

Please enter your CPAN site: []http://www.cpan.org
</code>

如果下载不了，可以设置cpan代理：
<code>
cpan> o conf http_proxy http://172.21.2.33:808

cpan> o conf commit

cpan> reload index
</code>

在bugzilla文件夹里
<code>
./checksetup.pl

/usr/bin/perl install-module.pl –all
</code>

如需安装apxs，用命令sudo apt-get install apache2-threaded-dev

=====5. 配置Bugzilla=====

修改localconfig：
<code>
$index_html = 1;
$db_pass = '123';
$webservergroup = 'www-data';
</code>

=====6. 修改Apache配置=====
sudo vim /etc/apache2/sites-available/default

<code>
<Directory "/var/www/bugzilla/">
        AddHandler cgi-script .cgi
        Options +Indexes +ExecCGI +FollowSymLinks
        DirectoryIndex index.cgi
        AllowOverride None
        Order allow,deny
        Allow from all
</Directory>
</code>

=====7. 重启Apache和MySQL=====
<code>
sudo /etc/init.d/apache2 restart
sudo /etc/init.d/mysql restart
</code>

=====8. 汉化=====

解压bugzilla-cn-4.4.zip到bugzilla目录下的template
