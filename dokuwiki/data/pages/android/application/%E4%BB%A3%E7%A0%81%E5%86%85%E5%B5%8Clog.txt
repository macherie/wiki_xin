====== 代码内嵌LOG ======

1、实现
<code java>
    private static final String LOG_TAG = "---MainActivity---" ;
    private static final boolean sDebug = !"user".equals(android.os.Build.TYPE);

    private static void sLogD(String msg) {
        if(sDebug) {
            android.util.Log.d(LOG_TAG, msg);
        }
    }
</code>

2、调用
<code java>
    sLogD("This is behavior. You can do it!");
</code>
