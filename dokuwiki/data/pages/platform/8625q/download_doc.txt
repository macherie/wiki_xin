======QPST烧录说明======
=====一.安装QPST后，选择eMMC Software Download工具=====
{{:playground:8625q:未命名.jpg|}}
=====二．空的EMMC（未烧录过版本），先下载bootloader： （如果是版本升级，请直接跳过此步骤）=====
测试机连接USB线，COM xx Phone in Download Mode端口出现，表明设备已经连接好了。

{{:playground:8625q:boot.jpg|}}

  - 1只有Program Boot Loaders 打勾，其他勾掉。
  - 2Flash Programmer file name 分别填写：MPRG8X25Q.hex，8X25Q_msimage.mbn。
  - 3Search path 2, 选择要下载文件的路径。
  - 4点击Download按钮，大约要一分钟

=====三.烧录其他image=====
如果 COM xx Phone in Download Mode 端口出现，表明设备已经连接好了，处在Download Mode；如果不在Download Mode ，点击Switch device to DLOAD…

{{:playground:8625q:image.jpg|}}

  - 1Program Boot Loaders勾掉，Program MMC device 打勾。
  - 2会出现Qualcomm MMC… usb mass storage的设备，必须选中这一行。
  - 3Load XML def.. 选择 rawprogram0.xml，Load patch def … 选择patch0.xml。
点Download，开始下载，出现write complete 后，下载完毕，按reset键3至5秒重启设备。
